# 禁忌参考清单（Contraindication Reference）

> **Skill 用途**：本文件为 `safety-agent` 提供中医用药禁忌的参考知识库，用于 `check_contraindications` 工具的执行依据。涵盖妊娠禁忌、儿童用药、肝肾功能不全用药风险、药食禁忌及线上平台操作边界。发现禁忌时必须警示，不可隐瞒。

---

## 1. 核心原则

1. **发现禁忌必须警示**：任何检测到的禁忌信号，必须明确告知患者，不可为了"不吓到患者"而隐瞒。
2. **线上不开具含禁忌药物的建议**：即使禁忌程度较轻，线上平台也不应给出涉及高风险禁忌药物的具体处方建议。
3. **禁忌不代表完全禁止治疗**：禁忌的含义是"需要专业会诊和谨慎处理"，而不是"无法治疗"，应引导患者寻求专业线下诊治。
4. **不确定时按禁忌处理**：如无法确认患者是否属于特殊人群（如未确认是否怀孕），按存在禁忌处理，优先保守。
5. **禁忌检查必须主动**：`safety-agent` 必须根据 `special_population_tags` 和 `current_medications` 主动扫描禁忌，不依赖患者自报。

## 2. 妊娠禁忌（Pregnancy Contraindications）

妊娠期用药禁忌是所有禁忌中最严格的类别。对 `pregnant_confirmed` 或 `pregnant_unconfirmed` 标签的患者，任何涉及以下药物大类的建议必须禁止。

### 2.1 妊娠绝对禁忌（Absolute Prohibition）

以下药物在妊娠期**绝对禁止**，线上问诊不得以任何形式提及其使用：

#### 2.1.1 破血逐瘀类（峻烈活血药）

| 药物 | 原因 | 风险 |
|------|------|------|
| 三棱 | 破血行气消积 | 可致流产、胎儿损伤 |
| 莪术 | 破血行气止痛 | 同上 |
| 水蛭 | 破血逐瘀 | 强烈破血，严禁 |
| 虻虫 | 逐瘀破积 | 同上 |
| 穿山甲（炮甲）| 活血通经 | 可能促宫缩 |
| 乳香（大量）| 活血止痛 | 大量可促宫缩 |
| 没药（大量）| 活血散瘀 | 同上 |

#### 2.1.2 峻下逐水类

| 药物 | 原因 | 风险 |
|------|------|------|
| 巴豆 | 峻下冷积 | 剧毒泻药，严禁 |
| 大戟 | 泻水逐饮 | 毒性强，可致流产 |
| 芫花 | 泻水逐饮 | 毒性强，严禁 |
| 甘遂 | 泻水逐饮 | 毒性强，严禁 |
| 牵牛子 | 泻下逐水 | 可损伤胎元 |
| 商陆 | 逐水消肿 | 有毒，可致流产 |
| 芒硝（大量）| 泻热通便 | 大量可致宫缩 |

#### 2.1.3 有毒药物类

| 药物 | 原因 | 风险 |
|------|------|------|
| 附子 / 乌头（生品）| 含乌头碱，有毒 | 心脏毒性，胎儿毒性 |
| 草乌（生品）| 同上 | 同上 |
| 朱砂 | 含汞，重金属毒性 | 胎儿神经发育损害 |
| 雄黄 | 含砷，重金属毒性 | 胎儿毒性，致畸 |
| 轻粉 | 含汞 | 严禁 |
| 砒霜（信石）| 含砷 | 剧毒，严禁 |
| 蜈蚣 | 有毒，祛风止痉 | 可致流产 |
| 全蝎 | 有毒，息风止痉 | 可能致流产 |
| 斑蝥 | 有大毒，破血逐瘀 | 严禁，可致流产死胎 |

#### 2.1.4 通经堕胎类

| 药物 | 说明 |
|------|------|
| 麝香 | 开窍通络，有促宫缩作用，严禁 |
| 冰片（大量）| 开窍，可能影响胎儿 |
| 桃仁（大量）| 活血祛瘀，破血力强时慎用 |
| 红花（大量）| 活血通经，大量可致宫缩 |
| 牛膝（大量）| 引血下行，有堕胎记载 |
| 益母草（大量）| 活血调经，可致子宫收缩 |

---

### 2.2 妊娠慎用类（需严格监督下使用）

以下药物在妊娠期**慎用**，线上平台不应给出相关建议，需引导至线下专科评估：

- 活血化瘀类（轻度）：丹参、川芎、当归（大量）
- 行气破滞类：枳实、枳壳（大量）
- 苦寒峻泻类：大黄（大量）、番泻叶
- 利水渗湿类：泽泻、茯苓皮（大量利水）
- 芳香化湿类：苍术（大量）

---

### 2.3 妊娠期安全用药类（可在专科医师指导下使用）

以下为相对安全、中医常用的孕期药物，但线上问诊仍不应给出具体建议，仅供参考：

- 健脾益气：党参、白术、山药、大枣
- 安胎常用：杜仲、菟丝子、续断、砂仁
- 轻度清热：黄芩（安胎要药）、竹茹
- 润肠通便（轻度）：火麻仁

> ⚠️ **重要提示**：即使是"安全"类药物，妊娠期使用仍需专科医师开具，线上问诊不应给出处方建议。

---

### 2.4 妊娠禁忌触发时的系统行为

当检测到 `pregnant_confirmed` 或 `pregnant_unconfirmed` 标签时：

```
1. 调用 check_contraindications 工具，记录妊娠禁忌标记
2. 写入 ContraindicationFlag：
   - contraindication_id: "ci_pregnancy"
   - name: "妊娠期用药禁忌"
   - severity: "absolute"（绝对禁忌）或 "relative"（慎用）
   - reason: "患者处于妊娠期，中医问诊系统不提供具体用药建议"
3. 在输出中加注以下警示（不可省略）：
```

**必须输出的警示文本**：
> ⚠️ **妊娠期用药警示**：您目前处于妊娠期（或可能怀孕），中医用药对孕妇有严格限制，部分中药对胎儿有潜在风险。本平台无法为您提供具体的中药建议，请务必前往具有妊娠期中医诊治资质的医疗机构，由专科医师面诊评估后再决定用药方案。

## 3. 儿童用药注意事项（Pediatric Medication Cautions）

### 3.1 儿童用药基本原则

- **剂量必须按体重/年龄调整**：成人剂量不适用于儿童，线上问诊不应给出成人剂量方案
- **优先选用儿童中成药**：有儿科专用品种时，优先推荐（仅建议，不开具）
- **部分药物有年龄下限**：部分药物对特定年龄段儿童禁止使用

### 3.2 儿童绝对禁忌药物

| 药物 / 成分 | 禁用年龄段 | 原因 |
|------------|-----------|------|
| 含朱砂成分的中成药 | 所有儿童 | 汞毒性，影响神经发育 |
| 含雄黄成分的中成药 | 所有儿童 | 砷毒性 |
| 附子、乌头（生品及大剂量）| 所有儿童 | 毒性风险高，代谢能力弱 |
| 大黄（大量）| < 6 岁 | 可致严重腹泻脱水 |
| 麝香类药物 | 新生儿/婴儿 | 神经系统影响 |
| 牛黄类（大量）| < 3 岁 | 需谨慎 |
| 酒制类药物（药酒）| 所有儿童 | 含酒精，儿童代谢差 |

### 3.3 儿童剂量参考原则

| 年龄段 | 成人剂量比例（参考） |
|--------|-----------------|
| < 1 岁 | 1/6 成人量 |
| 1–2 岁 | 1/4 成人量 |
| 3–5 岁 | 1/3 成人量 |
| 6–9 岁 | 1/2 成人量 |
| 10–14 岁 | 2/3 成人量 |
| ≥ 15 岁 | 成人量 |

> **重要**：以上为粗略参考，实际儿童用药剂量必须由儿科医师根据体重、病情、具体药物确定。线上问诊不开具儿童中药处方。

### 3.4 儿童特殊用药场景

- **高热**：儿童高热不建议自行使用苦寒退热药（大黄、黄连），应线下就医
- **腹泻**：儿童腹泻脱水风险高，不建议自行用药，建议线下评估
- **咳嗽有痰**：婴幼儿排痰能力弱，含化痰药的中成药需谨慎，避免影响气道

## 4. 肝肾功能不全用药风险（Hepatic & Renal Insufficiency）

### 4.1 肝毒性中药（Hepatotoxic Herbs）

以下中药具有已知或疑似肝毒性，对慢性肝病（`chronic_liver_disease`）患者禁止在线上给出建议：

#### 4.1.1 明确肝毒性（禁止使用）

| 药物 | 毒性机制 | 说明 |
|------|---------|------|
| 何首乌（生首乌，大剂量）| 蒽醌类，可致药物性肝损伤 | 已有大量病例报告，严格限制 |
| 雷公藤 | 多种毒性成分 | 肝毒性、肾毒性、生殖毒性，需严格管控 |
| 川楝子（大量）| 川楝素 | 有肝毒性 |
| 千里光 | 吡咯里西啶生物碱 | 肝静脉闭塞病 |
| 菊三七（土三七）| 吡咯里西啶生物碱 | 肝小静脉闭塞综合征，严禁与正品三七混淆 |
| 艾叶（大量）| 侧柏酮 | 大量可致肝损伤 |
| 黄药子 | 薯蓣皂苷等 | 可致药物性肝炎 |
| 苍耳子（大量）| 苍术苷 | 肝毒性 |
| 望江南子 | 含毒蛋白 | 肝毒性 |

#### 4.1.2 疑似或潜在肝毒性（肝病患者慎用）

- 大量使用苦楝皮、蓖麻子、相思子
- 含吡咯里西啶生物碱的药材：款冬花（大量）、紫草（部分品种）
- 酒制药物（肝病患者对酒精代谢能力下降）

### 4.2 肾毒性中药（Nephrotoxic Herbs）

以下中药具有已知肾毒性，对慢性肾病（`chronic_kidney_disease`）患者禁止在线上给出建议：

#### 4.2.1 明确肾毒性（严格禁止）

| 药物 | 毒性机制 | 说明 |
|------|---------|------|
| 马兜铃 | 马兜铃酸（AAs）| 马兜铃酸肾病，不可逆性肾损伤 |
| 关木通（木通中的有毒品种）| 马兜铃酸 | 同上，已禁止销售 |
| 广防己 | 马兜铃酸 | 同上 |
| 青木香 | 马兜铃酸 | 同上 |
| 天仙藤 | 马兜铃酸 | 同上 |
| 寻骨风 | 马兜铃酸 | 同上 |
| 朱砂莲 | 马兜铃酸 | 同上 |
| 雷公藤 | 多种毒性成分 | 肾毒性明显 |
| 草乌（大量）| 乌头碱 | 肾毒性 |
| 斑蝥 | 斑蝥素 | 肾毒性强，剧毒 |

> ⚠️ **马兜铃酸注意**：马兜铃酸类药物在中国部分地区的中成药中仍有存在，需警惕患者自购含马兜铃酸成分的中成药。如患者提及相关药物，必须警示。

#### 4.2.2 肾病患者高钾风险

以下中药或食疗方案对 CKD 患者存在高钾风险（CKD 患者需限钾）：
- 含钾量高的中药：车前草（鲜品）、葛根（大量）
- 食疗方中的高钾食物：香蕉、橙子、土豆（大量使用时）

### 4.3 肝肾双毒性药物

以下药物同时具有肝肾毒性，对慢性肝病和慢性肾病患者均严格禁止：

| 药物 | 说明 |
|------|------|
| 雷公藤 | 肝毒性 + 肾毒性 + 生殖毒性 |
| 朱砂 | 汞毒性影响肝肾 |
| 雄黄 | 砷毒性影响肝肾 |
| 斑蝥 | 强毒，肝肾均受损 |

## 5. 常见药食禁忌（Drug-Food Interactions）

### 5.1 经典中药配伍禁忌（十八反、十九畏）

#### 5.1.1 十八反（绝对配伍禁忌）

以下药物两两相反，绝对不可同时使用：

| 类别 | 反药对 |
|------|------|
| 甘草反 | 甘草 ↔ 大戟、甘遂、芫花、海藻 |
| 乌头反 | 乌头（含附子、草乌）↔ 贝母（川贝、浙贝）、瓜蒌、半夏、白蔹、白及 |
| 藜芦反 | 藜芦 ↔ 人参、沙参（南北）、丹参、玄参、苦参、细辛、芍药（赤白）|

> 记忆口诀："本草明言十八反，半蒌贝蔹及攻乌，藻戟遂芫俱战草，诸参辛芍叛藜芦。"

#### 5.1.2 十九畏（相畏配伍，一般不宜同用）

| 药物对 | 说明 |
|--------|------|
| 硫磺 ↔ 朴硝（芒硝）| 相畏 |
| 水银 ↔ 砒霜 | 相畏 |
| 狼毒 ↔ 密陀僧 | 相畏 |
| 巴豆 ↔ 牵牛子 | 相畏 |
| 丁香 ↔ 郁金 | 相畏（临床有争议，但一般建议回避）|
| 川乌、草乌 ↔ 犀角（水牛角）| 相畏 |
| 牙硝 ↔ 三棱 | 相畏 |
| 官桂（肉桂）↔ 赤石脂 | 相畏 |
| 人参 ↔ 五灵脂 | 相畏（临床有争议，但一般建议不同用）|

## 6. 线上平台明确禁止的建议类型

### 6.1 绝对禁止给出的内容

无论任何情况，以下内容在线上问诊平台中**绝对禁止**：

| 禁止内容 | 原因 |
|---------|------|
| 含剧毒药材的处方建议（砒霜、斑蝥、巴豆等）| 生命安全风险 |
| 含马兜铃酸类药物的建议 | 已知不可逆肾损伤 |
| 妊娠期任何中药处方建议 | 胎儿安全无法在线保障 |
| 儿童（<6岁）具体中药处方 | 剂量和安全性需专业评估 |
| 含朱砂、雄黄中成药的推荐（长期使用）| 重金属积累毒性 |
| 针对正在化疗/放疗患者的中药建议 | 复杂相互作用，需肿瘤科整合 |
| 外治类高风险操作的具体指导 | 见 6.2 节 |
| 任何形式的"根治""特效"表述 | 违反医疗广告规范 |
| 要求患者停用西药改用中药 | 超越平台能力边界，可能危及生命 |

---

### 6.2 外治类高风险操作禁止范围

以下外治操作在线上平台**禁止**给出具体指导：

| 操作类型 | 禁止原因 | 应引导至 |
|---------|---------|---------|
| 艾灸（特别是化脓灸、疤痕灸）| 操作不当可致烫伤感染 | 专业中医诊所 |
| 针刺（任何类型）| 涉及穴位定位和无菌操作，需执业资质 | 执业针灸医师 |
| 放血疗法（刺络）| 出血风险，感染风险 | 专业中医诊所 |
| 拔罐（走罐、留罐超 15 分钟）| 不当操作可致皮肤损伤 | 专业中医诊所 |
| 药物注射类（穴位注射）| 需执业资质，药物安全风险 | 医疗机构 |
| 直肠给药（中药灌肠）| 高风险操作，需专业指导 | 医疗机构 |
| 孕妇任何形式的艾灸 | 孕期禁忌穴位多，风险高 | 有妊娠期资质的中医科 |

**可以给出一般性建议的外治类型**（有限指导）：
- 简单穴位按摩（注明不适用于孕妇特定穴位）
- 泡脚方（温和药物，告知禁忌人群）
- 热敷/冷敷的一般建议（注明禁忌人群）

---

### 6.3 处方边界

线上中医问诊的处方边界如下：

| 能做的 | 不能做的 |
|--------|---------|
| 提供证型分析的候选（非确诊）| 开具具体中药处方（剂量、煎煮方法）|
| 推荐就医科室 | 声称可以替代线下就医 |
| 提供一般性生活调养建议 | 针对特殊人群给出中药方案 |
| 提供常见证型的一般用药方向（科普性质）| 针对复杂病情给出用药建议 |
| 推荐患者询问医师的问题 | 给出需要执业资质才能做的操作指导 |

## 7. 禁忌检测执行逻辑

### 7.1 禁忌扫描触发时机

`safety-agent` 在以下时机调用 `check_contraindications` 工具：

| 时机 | 扫描范围 |
|------|---------|
| 特殊人群标签写入后 | 对应特殊人群的全部禁忌类别 |
| 用药史更新后（`current_medications` 变化）| 中西药相互作用扫描 |
| `safety_reviewed` 阶段开始时 | 全量禁忌扫描 |
| 任何建议输出前 | 快速检查建议内容是否涉及禁忌 |

---

### 7.2 禁忌扫描逻辑

```
Step 1：读取特殊人群标签
  pregnant_confirmed / pregnant_unconfirmed → 执行妊娠禁忌扫描（第 2 节）
  neonate / infant / toddler / child        → 执行儿童用药禁忌扫描（第 3 节）
  chronic_liver_disease                     → 执行肝毒性药物扫描（第 4.1 节）
  chronic_kidney_disease                    → 执行肾毒性药物扫描（第 4.2 节）
    ↓
Step 2：读取 current_medications
  包含抗凝药 → 执行活血类中药相互作用扫描（第 5.3.1 节）
  包含降糖药 → 执行降糖相互作用扫描（第 5.3.2 节）
  包含免疫抑制剂 → 执行免疫抑制剂相互作用扫描（第 5.3.3 节）
  包含心脏药物 → 执行心脏药物相互作用扫描（第 5.3.4 节）
    ↓
Step 3：如系统需要输出任何含中药的建议
  对建议内容中涉及的药物，检查是否在本文件各节禁忌列表中
    ↓
Step 4：生成 ContraindicationFlag 列表
    ↓
Step 5：确定禁忌严重程度
  absolute → 绝对禁忌，不得给出相关建议，必须警示
  relative → 相对禁忌，需警示并建议专科评估
  interaction → 药物相互作用，需警示并建议告知主治医师
```

---

### 7.3 ContraindicationFlag 写入规范

```python
class ContraindicationFlag:
    contraindication_id: str    # 唯一 ID，格式：ci_{uuid4_short}
    name: str                   # 禁忌名称，如 "妊娠绝对禁忌-破血类"
    category: str               # 枚举：pregnancy / pediatric / hepatic / renal /
                                #       drug_food / drug_drug / platform_boundary
    severity: str               # 枚举：absolute / relative / interaction / advisory
    affected_population_tag: str | None  # 相关特殊人群标签
    triggering_medication: str | None    # 触发的西药名称（药物相互作用时填写）
    reason: str                 # 禁忌说明（人类可读）
    evidence_refs: list[str]    # 相关 fact_id 或 population_tag
    recommended_action: str     # 建议动作：warn_patient / refer_offline / stop_suggestion
```

## 8. 禁忌发现时的患者沟通规范

### 8.1 绝对禁忌发现时（`severity: absolute`）

**必须明确告知，不可模糊处理**：

```
⚠️ 重要安全提示：

根据您的情况，以下用药方向存在安全风险：
[具体说明，如：您处于妊娠期，破血活血类中药对胎儿有潜在风险]

本平台无法为您提供相关用药建议。

建议您：
• 前往[科室，如：具有妊娠期中医诊治资质的中医科]进行面诊
• 向医师说明您的[特殊情况，如：孕周、既往用药史]
• 任何中药使用都须经医师评估后再决定

[如有急性症状，请参考本次问诊结尾的就诊建议]
```

---

### 8.2 药物相互作用提示（`severity: interaction`）

```
用药提示：

您正在服用[西药名]，如果未来需要中药治疗，
部分中药可能与您的[西药名]产生相互作用。

具体建议：
• 就诊时请主动告知接诊医师您正在服用[西药名]
• 不要在未通知主治医师的情况下自行添加中药
• 如使用中成药，请查看成分，避免含有[风险成分]的产品

这一提示不影响您继续进行本次问诊，仅供就诊时参考。
```

---

### 8.3 药食禁忌提示（科普性质）

药食禁忌属于科普建议，可在适当时机告知，语气轻松，无需过度警示：

```
小提示：
如果您接受中药调理，通常建议在服药期间注意以下饮食：
• [具体禁忌，如：服含参方剂期间，建议少吃萝卜]
• 服药期间减少生冷、油腻、辛辣食物
• 饮茶（浓茶）可能影响部分药物吸收，建议间隔 1–2 小时

具体用药期间的饮食注意，请向开具处方的医师确认。
```

## 9. 禁忌参考与其他 Skill 的联动

| 联动 Skill | 联动关系 |
|-----------|---------|
| `special-population-rules` | 特殊人群标签是禁忌扫描的主要触发条件；孕妇、儿童、肝肾病患者的禁忌规则来自本文件 |

## 附录：禁忌严重程度速查

| `severity` 值 | 含义 | 系统行为 |
|--------------|------|---------|
| `absolute` | 绝对禁忌，任何情况下不可给出建议 | 停止相关建议输出 + 强制警示患者 + 建议线下 |
| `relative` | 相对禁忌，需专科评估后方可使用 | 警示患者 + 建议专科就诊 + 不给具体建议 |
| `interaction` | 药物/食物相互作用，需告知主治医师 | 告知相互作用风险 + 建议告知医师 + 继续问诊 |
| `advisory` | 一般注意事项，科普性质 | 适当时机告知患者，不影响问诊流程 |
